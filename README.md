# 📊 AdSales Predictor

A Machine Learning web app that predicts product sales based on advertising budget across different channels.

## 🚀 Features

* Predict sales using:

  * 📺 TV Advertising Budget
  * 📻 Radio Advertising Budget
  * 📰 Newspaper Advertising Budget
* Built with Streamlit
* Uses Linear Regression Model

## 🧠 Tech Stack

* Python
* Streamlit
* Scikit-learn
* Joblib

## ▶️ Run Locally

```bash
streamlit run app.py
```

## 📌 Project Idea

This project helps businesses understand how different advertising channels impact product sales.
